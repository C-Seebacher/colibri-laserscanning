Startup

Switch on the Laser

  1. switch on chiller (black, lower switch)
  2. wait unitl 21° (15 minutes), switch on main (orange, upper switch)
  3. turn key on
  4. wait until Warmup 100% (see small display 20 min)
  5. start Spectra-Physics Mai Tai software or use Tools Maitai in the scan software (humidity error is normal when the laser was off a long time)

Switch on the Microscope

  1. switch on the imic-controller-box (OK: only the lowest green LED on the DSP)
  2. switch on the galvo controller (OK; Galvos all LED off, Galvocontroller all green LEDs on)
  3. Start ScanMicroscope.vi
  4. missing init the imic-motors!
  5. run Single
  6. change only once to no (down) and run (takes contionous images)
  7. open shutter: Tools Maitai - Shutter open
  8. adjust LUT: IMAQ Imaq-scale
  9. adjust high voltage to ~800V: Edit PMT-gain
  10. adjust z to "176": iMic z

Shutdown

  1. switch off high Voltage: Edit: PMT-gain
  2. switch off laser: Tools Maitai
  3. switch off the Maitai Key (Mainswitch and chiller should stay on)
  4. switch off iMic, Galvos, PMT-supply...
